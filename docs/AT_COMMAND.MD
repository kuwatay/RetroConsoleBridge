# RetroConsoleBridge AT CommandsRetroConsoleBridge uses a Hayes-style AT command interface for configuration and management.Commands can be entered from the USB-CDC console or from a network session while in command mode.---# General Commands## ATTest command processor.```textATOK```## ATIDisplay basic product information.```textATI```## ATI4Display detailed system information.```textATI4```## AT?Display command summary.```textAT?```---# Configuration Commands## AT&WSave current settings to flash memory.```textAT&W```## AT&FRestore factory defaults.```textAT&F```## ATZReload saved configuration.```textATZ```## AT&VDisplay current configuration.```textAT&V```## AT&V1Display configuration in classic modem format.```textAT&V1```---# Wi-Fi Commands## AT$SSIDSet or query Wi-Fi SSID.Set:```textAT$SSID=MyWiFi```Query:```textAT$SSID?```## AT$PASSSet Wi-Fi password.```textAT$PASS=SecretPassword```## ATC1Connect to Wi-Fi.```textATC1```## ATC0Disconnect from Wi-Fi.```textATC0```## ATIPDisplay network information.```textATIP```---# Serial Port Commands## AT$SBSet or query UART baud rate.Examples:```textAT$SB=9600AT$SB=115200```Query:```textAT$SB?```---# Network Commands## AT$SPSet or query TCP server port.Example:```textAT$SP=23```Query:```textAT$SP?```Default:```text23```## ATTERMSelect terminal protocol mode.Values:```textATTERM=RAWATTERM=TELNET```Query:```textATTERM?```### RAWRecommended for console access and file transfer.### TELNETTELNET command negotiation enabled.## ATPASSWDSet or clear the remote access password.Set password:```textATPASSWD=mysecret```Clear password:```textATPASSWD=```Query:```textATPASSWD?```For security reasons, the actual password is not displayed.---# Console Commands## ATCONSOLESelect console ownership policy.Values:```textATCONSOLE=AUTOATCONSOLE=USBATCONSOLE=WIFI```Query:```textATCONSOLE?```### AUTOUSB console is active until a network client connects.### USBUSB console always owns the command interface.### WIFINetwork console always owns the command interface.## ATOReturn to online mode.```textATO```## ATHDisconnect the current network session.```textATH```---# S-Registers## ATS0Auto-answer setting.```textATS0=0```Currently reserved for future use.## ATS12Escape sequence guard time.```textATS12=50```Default:```text50```The escape sequence is:```text+++```---# Notifications## ATNOTIFYEnable or disable status notifications.Enable:```textATNOTIFY=1```Disable:```textATNOTIFY=0```Query:```textATNOTIFY?```Examples of notifications:```text[TELNET CONNECTED][TELNET DISCONNECTED][CONSOLE USB][CONSOLE WIFI]```---# Typical Configuration Example```textAT$SSID=MyWiFiAT$PASS=MyPasswordAT$SB=115200AT$SP=23ATTERM=RAWATCONSOLE=AUTOAT&WATC1```---# Recommended Client SoftwareRAW mode:```bashsocat -,raw,echo=0 tcp:<ip-address>:23```TELNET mode:```bashtelnet <ip-address> 23```For interactive console access and file transfer protocols such as XMODEM and Kermit, RAW mode is recommended.
